# 🤖 AI Resume Screening System

## 📌 Overview
This project is an AI-based Resume Screening System that predicts whether a candidate is **Selected** or **Rejected** based on resume text.

It uses Machine Learning and NLP techniques and also provides a simple explanation for its prediction.

---

## 🎯 Problem Statement
Recruiters receive a large number of resumes and manually filtering them is time-consuming.

---

## 💡 Solution
Built an AI system that:
- Takes resume text as input
- Predicts selection outcome
- Explains the reason based on key skills

---

## ⚙️ Features
- Resume classification (Selected / Rejected)
- NLP-based text processing
- Model accuracy evaluation
- Interactive user input
- Explanation of prediction (keywords)

---

## 🛠️ Tech Stack
- Python
- Pandas
- Scikit-learn
- NLP (CountVectorizer)
- Matplotlib (for visualization)

---

## 📊 Output Example
Prediction: Selected  
Reason: Contains keywords like Python, Machine Learning
